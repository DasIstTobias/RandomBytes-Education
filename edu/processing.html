Processing

Processing ist eine freie Grafikbibliothek und Programmiersprache.
Ziel der Sprache ist es, Nicht-Programmierern die Grundlagen der Programmierung auf einem visuellen Weg beizubringen.
Processing basiert auf der Programmiersprache Java und ist um zusätzliche Klassen, vereinfachte mathematische Funktionen und weitere Operationen ergänzt.
