Git

Git ist ein Versionskontrollsystem, das es Gruppen ermöglicht, effizient gemeinsam an Software zu arbeiten.
Es wurde ursprünglich von Linus Torvalds entwickelt und zeichnet sich durch seine Flexibilität, Leistungsfähigkeit und Sicherheit bei der gemeinsamen Codebearbeitung aus.
