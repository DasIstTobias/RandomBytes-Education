Arrays

Arrays kann man sich wie Variablen mit vielen Speicherwerten vorstellen. Ein Array erstellt man mit "int[]". Anschließend folgt der Name des Arrays. Nach einem Gleichheitszeichen folgt "new int[]", da ein neues Array erstellt werden soll. In die eckigen Klammern kommt dann die Anzahl, der Speicherplätze, die das Array haben soll. 


int[] einArray = new int[4];

Nun haben wir ein Array mit 4 Plätzen erstellt. Allerdings ist es leer. Die nummerierung der Plätze geht dabei von 0 aus. Das heißt ein Array mit 4 Plätzen hat keinen Platz 4. Ein Array mit 4 Plätzen hat die Plätze 0, 1, 2 und 3. Diese Plätze können wir nun mit Werten befüllen. das geht, wie bei Variablen, nur das bei Arrays noch die Nummer des Platzes in Eckigen Klammern angegeben wird. 


int[] einArray = new int[4];
einArray[0] = 175;
einArray[1] = -80;
einArray[2] = 62;
einArray[3] = 182;

Weil das aber nicht gut aussieht, können wir das Array auch gleich beim erstellen befüllen. Allerdings müssen wir dann auch alle Stellen gleich mit einem Wert belegen. Wir können kein Array mit 500 Plätzen erstellen, wenn wir nur die ersten 3 Werte beim erstellen definieren. Wenn wir die Werte beim erstellen Definieren, hat das Array immer automatisch genau so viele Plätze, wie wir Werte angegeben haben. 


int[] zahlen = {175, -80, 62, 182};

Dieses Array hat nun 4 Plätze, sodass wir keinen weiteren befüllen können. Wenn wir nun Beispielsweise ein Array mit 500 Plätzen erstellen wollen und nur die ersten 3 definieren wollen, geht das beispielsweise so. 


int[] einArray = new int[500];
einArray[0] = 175;
einArray[1] = -80;
einArray[2] = 62;

Die undefinierten Plätze stehen alle auf 0. Wenn wir gleich am Anfang ein Array mit ausrechend Plätzen erstellen, müssen wir uns später keine Sorgen um den Speicherplatz machen und haben nicht so viele verschiedene Variablen, dass es unübersichtlich wird.

Mit Arrays kann man wie mit Variablen arbeiten. Wir können beispielsweise mit ihnen rechnen. 


int[] einArray = new int[3];
einArray[0] = 175;
einArray[1] = -80;

einArray[2] = einArray[0] + einArray[1];

print(einArray[2]);

Die eigenschaften von Arrays kann man mit "arrayName.length" abrufen. 


int[] einArray = new int[500];
print(einArray.length);

Wenn wir keine Zahl in eckigen Klammern dahiter angeben, können wir alle Zahlen des Arrays auf einmal ausgeben. 


int[] einArray = new int[4];
einArray[0] = 175;
einArray[1] = -80;
einArray[2] = 62;
einArray[3] = 182;

print(einArray);

Das Array mit "int[]" ist natürlich nur ein Beispiel. Arrays können mit fast allen Variablentypen erstellt werden. Dementsprechend können wir für Zahlen mit Nachkommastellen auch ein "float" Array erstellen. 


float[] einFloatArray = new int[500];
