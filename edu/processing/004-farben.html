Farben

Im auf der letzten Seite haben wir am Ende Pacman gezeichnet. Dieser braucht natürlich auch Farbe. Dies ist aber sehr einfach. Zum Ausfüllen mit Farben muss man nur die Farbe in RGB angeben, nachdem man den Befehl "fill" benutzt hat. 


fill(255, 255, 0);
arc(50, 50, 80, 80, radians(30), radians(330));

Die Farbe von Text wird ebenfalls mit "fill" geändert. 


fill(255, 0, 0);
text("Moin", 50, 50);

Die Farbe kann jederzeit geändert werden, sodass verschiedene Formen in verschiedenen Farben existieren können. 


fill(255, 255, 0);
arc(50, 50, 80, 80, radians(30), radians(330));

fill(0, 255, 255);
rect(40, 40, 20, 20);
ellipse(80, 50, 20, 20);

Des Weiteren können die Farben auch nur mit einem Wert angegeben werden, welcher ebenfalls von 0 bis 255 geht. Dadurch kann man die Graustufe verändern. 


fill(160);
rect(20, 20, 60, 60);

Transparenz kann auch erzeugt werden, indem man einen vierten Wert von 0 bis 255 angibt. 


fill(100, 230);
rect(40, 40, 40, 40);

fill(0, 255, 0, 160);
rect(20, 20, 40, 40);

Mit "noFill" kann die Füllfarbe entfernt werden. 


noFill();
rect(20, 20, 60, 60);

Mit "stroke" und "strokeWeight" kann die Umrandung separat von der Füllfarbe eingestellt werden. 


strokeWeight(8);
stroke(0, 0, 255);
ellipse(50, 50, 60, 60);

Mit "noStroke" kann die Umrandung entfernt werden. 


noStroke();
ellipse(50, 50, 60, 60);

Die Hintergrundfarbe kann mit "background" geändert werden. 


background(255, 0, 0);
rect(20, 20, 60, 60);
