Variablen

In Processing gibt es natürlich auch Variablen. Das einfachste sind Integer, welche eine ganze Zahl angeben. Variablen können als Werte in den meisten Befehlen eingesetzt werden. Zur Veranschaulichung nehmen wir hier erstmal "print". 


int irgendeinName;
irgendeinName = 50;
print(irgendeinName);

Nachdem man mit int und dem Namen der Variable die Variable erstellt hat, kann man sie mit dem Namen der Variable definieren. Mit "=" kann man sie sofort auf einen Wert setzen, ansonsten kann man aber auch mit ihnen Rechnen. Beim Rechnen setzt man zuerst den Namen der Variable und dann wie beim setzen eines Wertes ein "=". Hinter das "=" kommt anschließend die Rechnung, bei der auch die Variable selbst genannt werden darf. 


int irgendeinName;
irgendeinName = 50;
irgendeinName = irgendeinName - 20;
print(irgendeinName);

Aber es gibt nicht nur ganze Zahlen. Mit "float" für "floating point number" kann man Gleitkommazahlen in Variablen speichern. Kommazahlen werden allerdings mit einem Punkt statt einem Komma geschrieben. 


float andereZahl;
andereZahl = 3.7;
print(andereZahl);

"float" speichert die Zahlen in 32-bit und kann daher nur maximal 8 Stellige Zahlen speichern. Sollte das nicht ausreichen, kann man "double" verwenden. "double" speichert Zahlen in 64-bit und kann daher bis zu 16 Stellige Zahlen speichern. 


double langeZahl;
langeZahl = 3.7777777777777;
print(langeZahl);

Manchmal braucht man aber auch einfach nur ja oder nein. Statt eine Variable zu haben, welche man immer zwischen 0 und 1 wechselt, kann man aber auch einfach eine boolean Variable nehmen. Diese kann entweder auf "true" oder "false" stehen. Mit dem Befehl "boolean" kann man eine boolean Variable erstellen. 


boolean jaOderNein;
jaOderNein = false;
print(jaOderNein);

Beim Programmieren kann es sich aber auch lohnen, ein Programm möglichst kompakt zu schreiben. Daher kann man das erstellen und definieren der Variable auch gleich in einem machen. 


int zahl = 50;
print(zahl);

Im aktiven Modus müssen die Variablen ganz am Anfang definiert werden, außerhalb von Schleifen. Nur dann können sie überall verwendet werden. Wenn eine Variable in "draw", "setup" oder einer anderen Umgebung definiert wird, kann sie nur in der jeweiligen Umgebung genutzt werden. 


int zahl = 60;

void setup() {
   println(zahl);
}

void draw() {
   println(zahl);
}

So kann man die Variable "zahl" überall nutzen.

Neben Zahlen und Ja/Nein als Variable, kann man aber auch einzelne Zeichen oder Zeichenketten speichern. Ein "char" ist dabei ein einzelnes Zeichen, während eine Zeichenkette von "String" erstellt wird. Den Wert, beziehungsweise die Zeichen schreibt man in Anführungszeichen. 


char einZeichen;
einZeichen = "A";
print(einZeichen);

  

    
    
  

      

String zeichenKette;
zeichenKette = "Hier kann was stehen!";
print(zeichenKette);

In Processing gibt es auch Systemvariablen, wie "mouseX" und "mouseY". Die Höhe und Breite des Gesamten Fensters kann mit "height" und "width" abgerufen werden. 


size(300, 300);
ellipse(height/2, width/2, 50, 50);

Dieses Programm malt einen Kreis genau in die Mitte des Zeichenbereiches, egal welche Größe bei "size" eingestellt ist.
