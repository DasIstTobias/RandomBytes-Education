Statischer und Aktiver Modus

In Processing gibt es zwei Formate zu Programmieren. Zum einen den Statischen Modus und zum anderen den Aktiven Modus. Beim Statischen Modus werden die Befehle des Codes nacheinander abgearbeitet, anschließend endet das Programm. 


println("1");
println("2");
println("3");
print("Ende");

Im aktiven Modus gibt es Funktionen, in die der Code kommt. Wir arbeiten hier vorerst nur mit "setup" und "draw". Zuerst werden die Sachen in "setup" einmal ausgeführt, anschließend werden die Sachen in "draw" immer wieder ausgeführt, ohne ein festes Ende. 


void setup() { // Alles hier wird zuerst einmal ausgeführt.
   println("1");
   println("2");
}

void draw() { // Das hier wird dann ganz oft wiederholt.
   println("Und ganz viele andere Zahlen.");
}

Da der Aktive Modus schon einmalig funktionen mit "setup" ausführen kann, funktioniert es nicht, wenn man Aktiv und Statisch mischt. 


void setup() {
   println("Aktiver Modus");
}

println("Statischer Modus");

Das geht also nicht.
"draw" wird normalerweise 60 Mal pro Sekunde ausgeführt. Wenn man diese Zahl verändern möchte, geht das mit "frameRate" und der Zahl der Aufrufe pro Sekunde, welche du haben möchtest. Wenn du also zum Beispiel 120 Aufrufe pro Sekunde willst, machst du folgendes. 


void setup() {
   frameRate(1);
   println("1");
   println("2");
}

void draw() {
   println("Und ganz viele andere Zahlen.");
}
