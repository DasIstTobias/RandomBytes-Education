If und Else-If Schleifen

Damit ein Programm auf Aktionen des Nutzers gut reagieren kann, muss man ihm sagen, was er machen soll, wenn etwas bestimmtes passiert. Dies lässt sich am besten im Aktiven Modus veranschaulichen. Wir benutzen nun If Schleifen.
Eine if Schleife erstellt man mit dem "if" Befehl. Als Argument nimmt man etwas, was diese Schleife auslösen soll. 


int x = 0;

void draw() {

  if (x == 5) {
    background(255, 0, 0);
  }
}

void keyPressed() {
  x++;
}

In diesem Programm setzen wir zuerst x auf 0. Wenn eine Taste gedrückt wird, erhöht sich x um 1. Der Fokus liegt hier aber auf der if Schleife in "draw". Diese If Schleife fragt immer wieder ab, ob x jetzt endlich auf 5 ist. Wenn ich 5 ist, wird der Hintergrund rot übermalt. Hier werden im Argument der if Schleife, zwei "=" genommen. Das liegt daran, dass wir x nicht definieren, sondern abfragen, wobei "==" und nicht "=" verwendet wird.

Wir können aber auch definierne, was passiert, wenn x nicht gleich 5 ist, also was in einem anderen Fall passiert. Das geht mit "else", was genau hinter die if Schleife geschrieben wird. 


int x = 0;

void draw() {

  if (x == 5) {
    background(255, 0, 0);
  } else {
    background(230);
  }
}

void keyPressed() {
  x++;
}

Nun übermalt das Programm den Hintergrund weiß, immer wenn x nicht 5 ist.

Dazwischen kann man beliebig viele weitere Argumente hinzufügen. Das geht mit "else if". 


int x = 0;

void draw() {

  if (x == 5) {
    background(255, 0, 0);
  } else if (x == 6) {
    background(0, 0, 255);
  } else {
    background(230);
  }
}

void keyPressed() {
  x++;
}

Nun übermalt das Programm den Hintergrund rot, wenn x gleich 5 ist und es übermalt den Hintergrund blau, wenn x gleich 6 ist. Wenn x irgendetwas anderes ist, wird der Hintergrung weiß übermalt.

Bei solchen Bedingungen gibt es natürlich nicht nur "==", sondern auch anderes. Beispielsweise:

  • ">=" für größer oder gleich.
  • "<=" für kleiner oder gleich.
  • ">" für größer.
  • "<" für kleiner.
  • "&&" für und. Dies kann zwischen zwei Bedingungen geschrieben werden, welche dann beide erfüllt sein müssen.
  • "||" für oder. Dies kann auch zwischen zwei Bedingungen geschrieben werden, von denen dann eine erfüllt sein muss.
